# Toprak Gayrimenkul

Sivas merkezli gayrimenkul isletmeniz icin finansal yonetim paneli.

## Ozellikler

- Gelir ve Gider Takibi
- Portfoy Yonetimi
- e-Fatura ve e-Arsiv
- Cari Hesap Yonetimi
- Detayli Raporlar
- 5 Tema ve Kisisellestirme

## Indir

[Son surumu indir](https://github.com/replitorge-lang/toprak-gayrimenkul/releases/latest)

## Gereksinimler

- Windows 10/11 (64-bit)
- ~240 MB disk alani

## Kurulum

`.exe` dosyasini calistirin ve sihirbazi takip edin.
